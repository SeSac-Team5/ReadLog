### #2 하단 TabBar 네비게이터 (`navigation/index.tsx`)

단일 Stack → **Bottom Tab + 4개 중첩 Stack** 구조로 전환.

```
MainTabs (TabBar)
├── HomeTab    → HomeStack (HomeScreen)
├── LibraryTab → LibraryStack (MyLibrary → BookDetail → …)
├── GroupTab   → GroupStack (GroupList → GroupHome → …)
└── MyPageTab  → MyPageStack (MyPage → EditProfile → …)
```

- `@react-navigation/bottom-tabs` 패키지 신규 설치
- 비로그인 스택(Login, SignUp) 구조는 동일 유지

### BUG-02 · bcrypt / passlib 버전 비호환 → 회원가입 500 에러

| 항목 | 내용 |
|---|---|
| 증상 | `POST /api/auth/signup` → 500 Internal Server Error, "가입 실패" Alert |
| 원인 | 시스템에 `bcrypt 5.0.0` 설치됨. `passlib 1.7.4`는 `bcrypt 4.x` 이상과 호환되지 않음. bcrypt 4.0에서 `__about__` 모듈 제거 → passlib 버전 감지 실패 → `ValueError: password cannot be longer than 72 bytes` |
| 수정 | `bcrypt 3.2.2` 다운그레이드 (`pip install bcrypt==3.2.2`) |
| 파일 | `requirements.txt`에 `bcrypt==3.2.2` 고정 추가 |
| 비고 | **백엔드 재시작 필수** — uvicorn 재기동 후 적용됨 |

### BUG-03 · TabBar가 Android 하단 네비게이션 바와 겹침 (Redmi Note 7+)

**1차 시도**: `useSafeAreaInsets` 적용 → `bottom: 0` 고정 (효과 없음)

| 원인 | `SafeAreaProvider`가 `App.tsx`에 없어 `useSafeAreaInsets`가 항상 0 반환 |
|---|---|
| 1차 수정 | `App.tsx`에 `SafeAreaProvider` 감싸기 |

**2차 시도**: `SafeAreaProvider` 추가 후에도 여전히 겹침

| 원인 | `bar` 스타일에 `height: 60` 고정 → `paddingBottom`이 탭 영역을 압축할 뿐 전체 높이가 늘어나지 않음 |
|---|---|
| 최종 수정 | `bar`에서 `height` 제거 → `tab`에 `height: 56` 이동. Android에서 `Math.max(bottom, 16)` 최솟값 보정 적용 (MIUI 기기 inset 0 보고 대응) |

### BUG-04 · MyLibraryScreen 목업 불일치 및 카드 크기 오류

**목업 대비 차이:**

| 항목 | 수정 전 | 수정 후 |
|---|---|---|
| 열 수 | 2열 | 3열 |
| 카드 너비 계산 | `flex:1` + `minWidth/maxWidth` (FlatList에서 불안정) | `useWindowDimensions`로 픽셀 정확 계산 |
| 통계 배너 | 단순 텍스트 행 | deepGreen 카드 + 완독 수 + 진행바 |
| 필터 칩 | 없음 | 전체 / 읽는 중 / 완독 수평 스크롤 |
| 카드 내 진행바 | 있음 (목업에 없음) | 제거, 완독 배지(✓)로 대체 |
